# GitHub User Fetcher

A simple Python application that uses the GitHub API to fetch and display public information about a GitHub user.

## Description

This project takes a GitHub username as input from the user and sends a request to the GitHub API. The application then retrieves the user's public profile information and displays it in a clean and readable format through the console.

The project demonstrates the use of:
- APIs
- HTTP GET requests
- JSON data handling
- Python requests library
- Basic error handling

## Requirements

- Python 3.x
- requests

## Technologies Used

- Python 
- Requests Library
- GitHub API

## Installation

Install the required package using:

bash
pip install -r requirements.txt

## Usage

Run the program using:

bash
python main.py


Enter a GitHub username when prompted and the application will display the user's profile information.
