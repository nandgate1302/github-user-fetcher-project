# Contributing Guidelines

Follow the steps below to submit your solution for this task.

## 1. Fork the Repository

Click the **Fork** button at the top-right of this repository, or use the following link:

```text
https://github.com/mulearngectcr/github-user-fetcher-project
```

This will create a copy of the repository under your GitHub account.

---

## 2. Clone Your Fork

```bash
git clone https://github.com/<your-username>/github-user-fetcher-project.git
cd github-user-fetcher-project
```

---

## 3. Create a Folder with Your Name

Inside the repository, create a folder with your name and place your solution inside it.

Example:

```text
github-user-fetcher-project/
├── john-doe/
│   ├── main.py
│   ├── requirements.txt
│   └── README.md
```

You may also include:
- Screenshots
- Additional files
- Notes about your implementation

---

## 4. Commit Your Changes

```bash
git add .
git commit -m "Add submission - John Doe"
```

---

## 5. Push to Your Fork

```bash
git push origin main
```

---

## 6. Create a Pull Request

1. Open your fork on GitHub.
2. Click **Contribute** → **Open Pull Request**.
3. Submit the pull request to this repository.

### Pull Request Title Format

```text
Submission - Your Name
```

Example:

```text
Submission - John Doe
```

---

## Important Notes

- Create **only one folder** with your name.
- Do **not** modify or delete someone else's submission.
- Ensure your code runs before creating the Pull Request.
- Feel free to attempt the bonus challenges as well.

Happy coding! 🚀
